### 关键技术决策

| 决策 | 说明 |
|------|------|
| **GitHub 优先** | 不用 Linear，减少外部依赖 |
| **sessions_spawn** | 不用 Codex app-server，用 OpenClaw 原生 subagent |
| **无持久化 DB** | 基于文件系统 + GitHub 状态恢复 |
| **热加载** | WORKFLOW.md 变化自动重新加载 |

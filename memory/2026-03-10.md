## Karpathy's AgentHub Project (20:05)

**项目**: https://github.com/karpathy/agenthub
**状态**: 已关注，待深入研究

**核心概念**:
- "GitHub is for humans. AgentHub is for agents."
- 为 AI agent 群体设计的协作平台
- 无主分支、无 PR、无 merge，只有 sprawling DAG of commits
- Git bundles + SQLite + Go 二进制（极简架构）
- 消息板用于 agent 间协调

**首个用例**: 组织 autoresearch 项目，从"单个 PhD 学生 agent"扩展为"研究社区 agent 群体"

**与 Symphony 对比**:
- AgentHub: 更极简，仅 Git + 消息板，无复杂编排层
- Symphony: 基于 OpenClaw sessions_spawn + GitHub Issues

**启发**: 可以考虑简化 Symphony 设计，减少编排复杂度

## 🎯 推荐策略

### 方案 A：纯 AI 主导（1-2 周可出成果）
```
全国大学生数学建模竞赛 + 美国大学生数学建模竞赛
蓝桥杯
全国企业竞争模拟大赛
中译国青杯翻译大赛
```
**优势**：不需要硬件、不需要现场、AI 全程搞定
**人类配合**：只需报名账号、最终提交

### 方案 B：AI+ 人类协作（含金量更高）
```
挑战杯学术科技作品竞赛（AI 写论文 + 人类答辩）
中国国际大学生创新大赛（AI 写计划书 + 人类路演）
全国大学生电子设计竞赛（AI 设计 + 人类制作）
```
**优势**：A/B 级比赛，获奖含金量高
**人类配合**：需要现场环节

### 方案 C：广撒网（多参加几个）
```
数学建模类：3 个
编程类：2 个
商业计划类：2 个
翻译/写作类：2 个
```
**优势**：获奖概率高
**劣势**：时间分散

## ⏰ 时间敏感度

### 需要立即准备的（通常上半年截止）
- 挑战杯系列（3-5 月）
- 中国国际大学生创新大赛（6-8 月）
- 全国大学生数学建模竞赛（9 月）
- ACM-ICPC（区域赛 10-12 月）

### 下半年为主的
- 美国大学生数学建模竞赛（1-2 月）
- 蓝桥杯（11 月 - 次年 3 月）
- 全国大学生电子设计竞赛（7-8 月）

---

## ❓ 需要确认的问题

1. **你们的专业是什么？** → 筛选专业相关比赛
2. **团队有几个人？** → 筛选团队/个人赛
3. **有硬件/实验室条件吗？** → 筛选是否需要设备
4. **能出面答辩/路演吗？** → 筛选是否需要现场
5. **目标是什么？** 保研/奖学金/兴趣？

# 比赛参赛形式分析

> 哪些比赛 AI 容易参与？哪些有困难？

| 全国大学生能源经济学术创意大赛 | B | 论文 | **全程撰写** |
| 全国大学生基础医学创新研究暨实验设计论坛 | B | 论文 + 设计 | **论文撰写** |

**人类只需**：报名账号、最终提交、可能需要署名

## 🟡 线上考试但开卷/可查资料（AI 可辅助 ⭐⭐⭐⭐）

**特点**：有线上考试，但可以查资料、开卷

| 比赛名 | 级别 | 考试形式 | AI 能做什么 |
|--------|------|---------|------------|
| 全国大学生数学建模竞赛 | B | **3 天开卷**，提交论文 | **核心建模 + 论文** |
| 美国大学生数学建模竞赛 | D | **4 天开卷**，提交论文 | **全程辅助** |
| 全国大学生统计建模大赛 | B | 提交论文 + 分析 | **统计分析 + 论文** |
| 数维杯全国大学生数学建模竞赛 | D | 开卷，提交论文 | **建模 + 论文** |
| 五一数学建模竞赛 | D | 开卷 | **建模** |
| 中青杯全国大学生数学建模竞赛 | D | 开卷 | **建模** |
| 认证杯数学中国数学建模竞赛 | D | 开卷 | **建模** |
| 华中杯大学生数学建模挑战赛 | D | 开卷 | **建模** |
| 全国企业竞争模拟大赛 | B | 在线决策模拟 | **决策辅助** |
| 全国高等院校数智化企业经营沙盘大赛 | B | 在线经营模拟 | **经营决策** |
| 学创杯全国大学生创业综合模拟大赛 | B | 创业模拟 | **模拟决策** |
| 全国大学生电子商务创新创意及创业挑战赛 | B | 方案 + 答辩 | **方案撰写** |

**优势**：
- ✅ 开卷考试，可以查资料
- ✅ 数学建模类时间充足（3-4 天）
- ✅ 模拟决策类 AI 擅长
- ✅ 主要是提交作品

**注意**：
- ⚠️ 数学建模需要人类账号登录提交
- ⚠️ 模拟决策类可能需要人类操作界面

## 🟠 线上考试有监考（AI 参与困难 ⭐⭐）

**特点**：有摄像头监考、屏幕监控、人脸识别

| 比赛名 | 级别 | 监考情况 | AI 参与难度 |
|--------|------|---------|------------|
| ACM-ICPC | B | **摄像头监考 + 代码查重** | ❌ 困难 |
| 蓝桥杯全国软件和信息技术专业人才大赛 | B | **摄像头监考** | ❌ 困难 |
| 百度之星程序设计大赛 | B | 在线编程 + 监考 | ❌ 困难 |
| 中国大学生程序设计大赛 CCPC | D | 现场/线上监考 | ❌ 困难 |
| 全国大学生软件创新大赛 | C | 可能有监考 | ⚠️ 不确定 |
| 全国大学生软件测试大赛 | C | 在线测试 | ⚠️ 不确定 |
| 全国高校计算机能力挑战赛 | C | 在线考试 | ⚠️ 不确定 |
| 传智杯全国大学生 IT 技能大赛 | D | 在线编程 | ⚠️ 不确定 |
| 全国大学生信息安全竞赛 | B | 可能有 CTF 线上赛 | ⚠️ 不确定 |
| 华为 ICT 大赛 | B | 在线考试 + 实验 | ⚠️ 不确定 |
| 大唐杯全国大学生新一代信息通信技术大赛 | B | 在线考试 | ⚠️ 不确定 |

**难点**：
- ❌ 摄像头全程监控
- ❌ 屏幕共享/录制
- ❌ 人脸识别验证
- ❌ 代码查重系统
- ❌ 异常行为检测（切屏、长时间不看屏幕等）

**AI 能做的**：
- 赛前培训、刷题训练
- 赛后复盘分析
- 但不能在考试过程中参与

**特点**：必须到现场，有人脸识别、现场操作

| 全国大学生金相技能大赛 | B | 现场操作 |
| 全国大学生地质技能竞赛 | B | 现场操作 |
| 全国大学生测绘学科竞赛 | B | 现场测量 |
| 世界技能大赛 | B | 现场技能 |
| 全国职业院校技能大赛 | B | 现场操作 |

## 📊 推荐策略（按 AI 可参与度）

### 方案 1：纯作品提交（最推荐 ⭐⭐⭐⭐⭐）
```
✅ 挑战杯系列（A 级）
✅ 中国国际大学生创新大赛（A 级）
✅ 全国大学生广告艺术大赛（B 级）
✅ 全国大学生能源经济学术创意大赛（B 级）
✅ 各类设计大赛（B/C 级）
✅ 翻译大赛（C/D 级）
```
**优势**：无考试、无监考、AI 全程参与
**人类配合**：报名账号、最终提交

### 方案 2：开卷建模类（强烈推荐 ⭐⭐⭐⭐⭐）
```
✅ 全国大学生数学建模竞赛（9 月，B 级）
✅ 美国大学生数学建模竞赛（1-2 月，D 级）
✅ 全国大学生统计建模大赛
✅ 各类数学建模挑战赛
```
**优势**：开卷、时间长（3-4 天）、AI 强项
**人类配合**：账号登录、最终提交确认

### 方案 3：模拟决策类（推荐 ⭐⭐⭐⭐）
```
✅ 全国企业竞争模拟大赛
✅ 全国高等院校数智化企业经营沙盘大赛
✅ 学创杯全国大学生创业综合模拟大赛
```
**优势**：AI 擅长决策优化
**人类配合**：可能需要人类操作界面

### 方案 4：避开监考类编程比赛（不推荐 ⭐⭐）
```
❌ ACM-ICPC（有监考）
❌ 蓝桥杯（有监考）
❌ 百度之星（有监考）
```
**原因**：摄像头监考 + 代码查重，AI 无法参与考试过程

## ⚠️ 需要避坑的比赛

**有监考系统的编程比赛**（AI 无法参与考试过程）：
- ACM-ICPC
- 蓝桥杯
- 百度之星
- CCPC
- 各类在线编程竞赛

**需要现场操作的**：
- 电子设计竞赛（4 天封闭）
- 机器人大赛（现场对抗）
- 各类技能大赛（现场操作）

**需要现场演讲/答辩的**：
- 英语演讲比赛
- 辩论赛
- 创业大赛路演（虽然 AI 可以写稿，但需要人类出面）

## 💡 最佳策略

**组合参赛**：
```
2 个数学建模（国赛 + 美赛）
1 个挑战杯（学术科技或创业计划）
1 个创新大赛
1 个设计/广告类
1 个翻译/写作类
```

**时间分布**：
- 1-2 月：美赛
- 3-5 月：挑战杯
- 5-6 月：广告艺术大赛、统计建模
- 6-8 月：创新大赛
- 9 月：国赛建模

**预期获奖**：按这个组合，AI 全程深度参与，获奖率应该很高！

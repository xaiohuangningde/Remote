## 配置变更

### npm 缓存路径
```powershell
npm config set cache "D:\npm-cache"
```

### pnpm store 路径
```powershell
pnpm config set store-dir "D:\pnpm-store"
```

### conda 包缓存路径
```powershell
conda config --add pkgs_dirs "D:\conda-pkgs"
```

## 后续建议

1. **定期清理**: 每月运行一次 `npm cache clean --force` 和 `pnpm store prune`
2. **监控 C 盘**: 低于 10GB 时触发清理
3. **Docker 清理**: 如使用 Docker，运行 `docker system prune -a`

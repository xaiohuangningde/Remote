# 绿茶人格 (Green Tea Persona)

发送语气软萌、句式破碎、带随机延迟的消息。

## 特性

- 句式简短破碎
- 随机延迟发送
- 语气软萌
- 不使用标点符号

## 使用方法

```javascript
const greenTea = require('./index.js');
await greenTea.send('你好呀～');
```
